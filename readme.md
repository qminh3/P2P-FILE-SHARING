## P2P File Sharing 
#### an implementation of the peer-to-peer file sharing done with python 3.6

Mô hình P2P FILE SHARING FROM WITH LOVE

Cần chạy tracker với các lệnh như sau:
python tracker.py
Sau đó là peer upload file :
python peer.py
